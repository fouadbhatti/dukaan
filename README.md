## Prerequisites:

1.  [Install Docker CLI](https://docs.docker.com/install/)

## Run the application

In a terminal, run:

```
$ docker pull khizarali/store
$ docker run -d -p 8090:8090 --name store khizarali/store:ghq
```

You can now access the application at http://localhost:8090
